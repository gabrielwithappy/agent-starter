# List Plugins Command

Display all installed plugins and their skills.

## Usage

Run the list command to see all installed plugins:

```powershell
python .agent/skills/plugin-manager/scripts/manage.py list
```

This will show:
- Plugin name
- Repository URL
- Owner
- Installation timestamp
- List of skills included

## Example

```
/plugin:list
```
